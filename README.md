# Training Planner

**Автор:** Концевой Александр

## Описание

Training Planner — это графическое приложение на Python для планирования и отслеживания тренировок. Программа позволяет добавлять тренировки с указанием даты, типа и длительности, фильтровать записи по дате и типу, а также сохранять и загружать данные в формате JSON.

## Требования

- Python 3.6 или выше
- Tkinter (входит в стандартную библиотеку Python)
